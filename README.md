Modern e-commerce landing page for Spoofer & Cheats. My first major project built with Next.js App Router.

---

## Info

### Project Purpose
Custom-made landing page for a single client (MintzCheatz), focusing on high performance and clean "Glassmorphism" design.

### Tech Stack
| Tool | Usage |
| ---- | ----- |
| Next.js 14 | Framework & Routing |
| Tailwind CSS | Styling & Animations |
| Vercel | Deployment & Hosting |

### Features
* **Full Responsive Design** - Works on mobile, tablet, and desktop
* **Dynamic UI** - Smooth hover effects and modern dark theme
* **Optimized Images** - Blazing fast loading with Next/Image
