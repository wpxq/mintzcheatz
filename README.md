# MintzCheatz E-shop

This is my first project developed using the Next.js framework. It is a modern, high-performance, and responsive e-shop specialized in gaming software and spoofers, created for MintzCheatz.

## Features

* **Next.js Performance** – Optimized loading speeds utilizing the latest App Router and Turbopack features.
* **Interactive Snow Effect** – Custom visual component providing a unique atmospheric background.
* **Full Responsiveness** – Complete mobile and tablet compatibility for a seamless user experience across all devices.
* **Modern Dark UI** – A clean glassmorphism-based design with a dark theme and purple accents.
* **Product Management** – Structured product grid designed for clarity and user conversion.
* **Smooth Scrolling** – Fluid navigation implemented throughout the entire landing page.
* **SEO Implementation** – Configured metadata and keywords for improved search engine visibility.
* **Vercel Deployment** – Automated CI/CD pipeline integrated directly with the GitHub repository.

## Technical Stack

* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS
* **Deployment:** Vercel
* **Language:** TypeScript
